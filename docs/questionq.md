## 附录：相关数据表字段参考

### behavior_fact 表常用字段

| 字段                | 说明                                                 |
| ------------------- | ---------------------------------------------------- |
| user_id             | 用户ID                                               |
| event_time          | 事件时间                                             |
| event_name          | 事件名称（首页/商品详情/加入购物车/结算/购买完成等） |
| session_id          | 会话ID                                               |
| event_properties    | 事件属性（商品ID/商品名称/价格/数量等）              |
| session_device_type | 设备类型                                             |
| session_browser     | 浏览器                                               |
| session_city        | 城市                                                 |
| session_ad_source   | 广告来源                                             |

### user_dim 表常用字段

| 字段              | 说明           |
| ----------------- | -------------- |
| user_id           | 用户ID         |
| initial_datetime  | 首次访问时间   |
| initial_ad_source | 首次广告来源   |
| last_datetime     | 最后访问时间   |
| total_sessions    | 总会话数       |
| user_variant      | 用户自定义变量 |
